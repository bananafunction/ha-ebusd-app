# ha-ebusd-app
App for http://home-assistant.io

Created by @LukasGrebe.  Improved and maintained by @tim-devel

## How to Install
1. In Home Assistant OS go to the Settings > Apps section,
2. In the overflow menu click "Repositories"
3. Add `https://github.com/bananafunction/ha-ebusd-app`
4. Wait for the ebusd App to show up or click reload in the same overflow menu
5. Install eBUSd.
6. Follow Documentation, Setup and Install of the App


## [ebusd](https://github.com/bananafunction/ha-ebusd-app/tree/main/ebusd) 

This Add-on runs [ebusd](http://ebusd.eu), a daemon for handling communication with eBUS devices connected to a 2-wire bus system (“energy bus” used by numerous heating systems), in [Home Assistant OS](https://www.home-assistant.io/installation/raspberrypi). You can simply plugin an ebus hardware interface into your RaspberryPi, and install the app to run ebusd.
